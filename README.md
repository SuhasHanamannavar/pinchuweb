# Memoir Desktop — Showcase Website

A stunning landing page for Memoir Desktop, built with Next.js 16, Tailwind CSS, and Framer Motion.

## Features
- Animated gradient mesh background
- Glassmorphism cards
- Framer Motion scroll animations
- Responsive design
- Dark theme
- SEO optimized

## Deploy

### Vercel (Frontend)
1. Push to GitHub
2. Import on vercel.com
3. Deploy

### Render (Backend API)
1. Connect GitHub repo
2. Set build command: `npm install && npm run build`
3. Set start command: `npm start`

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
